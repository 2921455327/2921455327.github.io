利用生成器函数解决回调地狱的问题

2022-04-02

需求：1秒后控制台输出111，然后2秒后输出222，然后三秒后输出333

原始做法

setTimeout(() => {
    console.log(111);
    setTimeout(() => {
        console.log(222);
        setTimeout(() => {
            console.log(333);
        }, 3000);
    }, 2000);
}, 1000);

这只是三层嵌套，想象一下，如果有100层，这样的代码不仅臃肿，还不利于后期的管理
下面用生成器函数解决一下这个问题：

生成器函数做法

function one(){
    setTimeout(() => {
        console.log(111);
        result.next()
    }, 1000);
}
function two(){
    setTimeout(() => {
        console.log(222);
        result.next()
    }, 2000);
}
function three(){
    setTimeout(() => {
        console.log(333);
    }, 3000);
}
function * gen(){
    yield one()
    yield two()
    yield three()
}
let result = gen()
result.next()
